# Zettelkasten Note Taking Method

Created: 2025-01-12T08:00:00Z
Tags: productivity, note-taking, zettelkasten

---

The Zettelkasten method is a note-taking system developed by Niklas Luhmann. It emphasizes atomic notes and bidirectional [[20250110090000]] links between ideas.

Key principles:
1. One idea per note
2. Unique identifiers for each note
3. Index for discoverability
4. Links create context
