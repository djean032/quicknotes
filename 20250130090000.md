Researching TF-IDF (Term Frequency-Inverse Document Frequency) for the suggestion engine in [[20250115090000]] QuickNotes.

**TF (Term Frequency):**
- How often a word appears in a document
- Usually normalized by document length

**IDF (Inverse Document Frequency):**
- log(total_docs / docs_containing_term)
- Rare words get higher weight

**Cosine Similarity:**
- Compare two document vectors
- Range: 0 (unrelated) to 1 (identical)

For implementation:
1. Build word index during indexing phase
2. Calculate TF-IDF vectors for each note
3. Compare query note against all others
4. Return top matches
